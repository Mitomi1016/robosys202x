# plusコマンド
![test](https://github.com/Mitomi1016/robosys2022/actions/workflows/test.yml/badge.svg)

標準入力から読み込んだ数字を足す。

## リポジトリについて
ロボシステムの授業で使うリポジトリです

## プログラムの説明
入力した整数を順番に足すことのできるプログラム


## 使い方
seq 8 | ./plusをターミナルに入力すると入力した整数までの数字をたす

## 実行例
seq 8 | ./plus

36.0

* Python
  * テスト済み: 3.7〜3.10
